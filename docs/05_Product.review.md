# [SHOOKING_STORE] 서비스 개발 방향성 및 기능 정의서

## 1. 문서 개요

본 문서는 요청해주신 이메일 내용과 디자인 시안(피그마)을 바탕으로, 앞으로 개발될 모바일 쇼핑몰의 주요 기능과 화면 구성을 정리한 문서입니다. 본격적인 개발에 들어가기 전, 원하시는 방향과 일치하는지 확인하기 위해 작성되었습니다.

---

## 2. 핵심 요청사항 정리

이메일을 통해 전달주신 핵심 목표를 다음과 같이 이해하고 개발을 진행할 예정입니다.

- **모바일 최적화:** PC보다 스마트폰 환경에서 고객이 더 편하게 쇼핑할 수 있도록 만듭니다.
- **빠른 속도:** 상품 사진이 많아도 버벅거리지 않고 부드럽게 화면이 넘어가도록 만듭니다.
- **간편한 구매:** 쇼핑 흐름이 끊기지 않도록, 상품을 구경하는 도중에 바로 장바구니에 담을 수 있게 만듭니다.

---

## 3. 화면 구성 및 동작 방식 (디자인 시안 분석)

전달받은 디자인 시안을 토대로 실제 구현될 화면의 모습을 정리했습니다.

### 3.1. 상품 목록 화면

- **보기 방식:** 스마트폰 화면에서 상품이 잘 보이도록 한 줄에 2개씩(2열) 배치합니다.
- **상품 정보:** 상품 사진, 브랜드, 이름, 가격, 할인율을 한눈에 알아볼 수 있게 표시합니다.
- **필터 및 정렬:**
  - 화면 상단에서 '신발', '의류' 등 카테고리를 좌우로 넘겨가며 선택할 수 있습니다.
  - '신상품순', '낮은 가격순' 등으로 상품 순서를 바꿀 수 있습니다.

### 3.2. 장바구니 담기 기능

- **바로 담기 버튼:** 상품 사진 오른쪽 아래에 있는 '장바구니 아이콘'을 누르면 상세 페이지로 이동하지 않고 바로 담을 수 있습니다.
- **옵션 선택 창 (하단 팝업):** 아이콘을 누르면 화면 아래에서 옵션(사이즈, 색상 등)을 선택하는 창이 부드럽게 올라옵니다.
- **완료 알림:** 장바구니 담기가 완료되면 "장바구니에 추가되었습니다."라는 안내 문구가 잠시 떴다가 사라집니다.

## 4. 주요 기능 상세 목록

| 구분          | 기능 이름      | 설명                                                                 | 비고                             |
| :------------ | :------------- | :------------------------------------------------------------------- | :------------------------------- |
| **상품 보기** | 상품 목록 표시 | 등록된 상품들을 고객 화면에 보여줍니다.                              | 스크롤을 내리면 계속 상품이 나옴 |
|               | 카테고리 선택  | 선택한 카테고리(신발, 의류 등) 상품만 골라서 보여줍니다.             |                                  |
| **장바구니**  | 옵션 선택      | 장바구니에 담기 전, 사이즈나 색상을 선택하는 기능입니다.             | 화면 하단에서 창이 올라옴        |
|               | 장바구니 저장  | 선택한 상품을 장바구니에 저장합니다.                                 |                                  |
| **기타**      | 화면 맞춤      | 스마트폰 기종이나 PC 화면 크기에 맞춰 레이아웃이 깨지지 않게 합니다. |                                  |
