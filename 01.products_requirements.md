# 상품 목록 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보

### 프로젝트명:

패션 쇼핑몰 상품 목록 페이지 (모바일)

### 사용하게 될 기술:

React 18(19)

### 문서 작성자: 서경진

## 📝 고객 요구사항 정리

1. 모바일 최적화
   ● 웹페이지 기반이나 모바일 환경에서의 사용 편의성을 최우선으로 개발
   요망
   ● 화면 비율이 맞지 않아 발생하는 고객 이탈 방지 필수

2. 상품 리스트 UI 개선
   ● 상품명보다 상품 사진(이미지) 위주의 배치 요청
   ● 신발 디테일이 잘 보이도록 사진을 강조하는 레이아웃 적용
   장바구니 기능 강화
   ● 장바구니 담기 시, 화면 상단 아이콘에 실시간 상품 개수 배지 표시 기능
   추가 (직관적 확인 용도)

3. 테스트 및 검수
   ● 내부 직원 피드백을 위해 테스트 URL 공유 요청

## 📋 기능 목록

### 1. 레이아웃 (Layout)

- **모바일 뷰포트 설정:** 모바일 기기 해상도 대응을 위한 메타 태그 및 CSS 설정
- **반응형 컨테이너:** 최대 너비(max-width) 제한을 둔 중앙 정렬 컨테이너 구현

### 2. 헤더 (Header)

- **`<Header />` 컴포넌트**
  - 스크롤 시 상단 고정 (Sticky)
  - 우측 상단 장바구니 아이콘 배치
  - **장바구니 배지(Badge):** `cartCount` 상태값에 따라 숫자 렌더링 (0일 경우 숨김 처리)

### 3. 상품 리스트 (Product List)

- **`<ProductList />` 컴포넌트**
  - **2단 그리드(Grid) 레이아웃:** CSS Grid를 활용하여 한 줄에 상품 2개씩 배치 (`grid-template-columns: 1fr 1fr`)
  - **이미지 비율 고정:** 상품 이미지가 잘리지 않도록 `aspect-ratio` 설정 및 `object-fit: cover` 적용

### 4. 상품 아이템 (Product Item)

- **`<ProductItem />` 컴포넌트**
  - 상품 이미지, 브랜드명, 상품명, 가격 표시
  - **장바구니 담기 버튼:** 클릭 시 전역 상태(장바구니 카운트) 업데이트 로직 연결 (`onClick` 이벤트)
